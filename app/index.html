HOW TO USE

File Support

Supports .mcr, .bin, .gme, .mcd, .srm memory card files.

Managing Saves

  • Copy Save: Drag and drop a save icon from one card to another.
  • Delete Save: Click the Delete button or select a slot and press Delete key.
  • Undelete: Deleted a save by mistake? Look for the "RECOVER" button on empty slots.

Import / Export

  • Export: Click the Export button to download a single save as .mcs.
  • Import: Click the Import button on any slot to import a .mcs file to that location.
